# Inter-VLAN Routing with DHCP Server on Router

## 📌 Project Overview

This project demonstrates Inter-VLAN routing and DHCP configuration using a router and multilayer switch in Cisco Packet Tracer.

## 🎯 Objective

* Configure VLANs
* Enable Inter-VLAN Routing
* Configure DHCP Server
* Assign IP Automatically
* Enable Communication Between VLANs

## 🧠 Network Design

### VLAN 10 – IT Department

Network: 192.168.1.0/24
Gateway: 192.168.1.1

### VLAN 20 – HR Department

Network: 192.168.2.0/24
Gateway: 192.168.2.1

### VLAN 30 – FIN Department

Network: 192.168.3.0/24
Gateway: 192.168.3.1

## 🖥️ Devices Used

* Router (2811)
* Multilayer Switch
* Access Switches
* PCs
* Cisco Packet Tracer

## 🔧 Features Implemented

* VLAN Configuration
* Trunk Configuration
* Router on Stick
* DHCP Configuration
* Inter-VLAN Communication
