# Configuring DHCP Server on Multilayer Switch

## 📌 Project Overview

This project demonstrates DHCP Server configuration using a Multilayer Switch in Cisco Packet Tracer.

The topology includes:

* Multilayer Switch (DHCP Server)
* 2 Access Switches
* 6 PCs
* 2 Departments (IT & FIN)

---

## 🧠 Network Design

### IT Department

* Network: 192.168.10.0/24
* Default Gateway: 192.168.10.1
* DHCP Range: 192.168.10.11 - 192.168.10.254

### FIN Department

* Network: 192.168.20.0/24
* Default Gateway: 192.168.20.1
* DHCP Range: 192.168.20.11 - 192.168.20.254

---

## 🎯 Objectives

* Configure Multilayer Switch as DHCP Server
* Enable Layer 3 Routing
* Create Multiple DHCP Pools
* Configure Inter-VLAN Communication
* Verify Network Connectivity

---

## 🖥️ Devices Used

* Multilayer Switch (3560)
* Access Switch (2960)
* PCs
* Cisco Packet Tracer

---

## 🔧 Features Implemented

* Layer 3 Switching
* DHCP Server on Switch
* Inter Network Communication
* IP Exclusion
* Automatic IP Assignment
