# DHCP SNOOPING CONFIGURATION

---

## Router Configuration (R1)

enable
conf t
interface g0/0
ip address 192.168.1.1 255.255.255.0
no shutdown
exit

---

## Switch Configuration (S1)

enable
conf t

ip dhcp snooping
ip dhcp snooping vlan 1

interface range fa0/1-5
ip dhcp snooping trust
exit

do show vlan
do wr

---
