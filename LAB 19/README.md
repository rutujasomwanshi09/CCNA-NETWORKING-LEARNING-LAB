Layer 3 Switch Routing Lab

📌 Project Title

Layer 3 Switch Routing Between Two Networks

---

🎯 Objective

This lab demonstrates:

- Layer 3 switching
- Inter-network communication
- IP routing on multilayer switch
- Router and Layer 3 switch connectivity

---

🏗️ Topology Overview

Devices Used:

- 1 Router (1941)
- 2 Multilayer Switch (3650)
- Network cables
