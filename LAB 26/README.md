# DHCP Server Configuration on Router (Cisco Packet Tracer)

## 📌 Project Overview

This project demonstrates DHCP Server configuration on a Cisco Router for multiple departments.

The topology consists of:

* 1 Router (DHCP Server)
* 2 Switches
* 6 PCs
* 2 Networks (IT & HR Department)

## 🧠 Network Details

### IT Department

* Network: 192.168.1.0/24
* Default Gateway: 192.168.1.1
* DHCP Range: 192.168.1.11 - 192.168.1.254

### HR Department

* Network: 192.168.2.0/24
* Default Gateway: 192.168.2.1
* DHCP Range: 192.168.2.11 - 192.168.2.254

## 🎯 Objective

* Configure Router as DHCP Server
* Assign IP addresses automatically
* Configure Multiple DHCP Pools
* Verify Connectivity using Ping

## 🖥️ Devices Used

* Cisco Router 2911
* Cisco Switch 2960
* PCs
* Cisco Packet Tracer

## 🔧 Features Implemented

* Multi Network DHCP
* IP Exclusion
* Automatic IP Assignment
* Inter Network Communication
