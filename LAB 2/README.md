# 🖧 Router Basic Configuration Lab (Cisco Packet Tracer)

## 📌 Overview
This lab demonstrates the basic configuration of a router using Cisco Packet Tracer. It includes setting up hostname, passwords, console and VTY access, security features, and system settings.

---

## 🎯 Objectives
- Configure basic router settings
- Secure console and remote access
- Set hostname and banner message
- Enable password encryption
- Configure login security features

---

## 🧰 Devices Used
- Router (Cisco 2911)
- PC (PC-PT)
- Ethernet Cable

---

## ⚙️ Configuration Tasks
- Set hostname
- Configure enable password
- Configure console and VTY passwords
- Set banner message
- Disable DNS lookup
- Set domain name
- Create local user
- Enable password encryption
- Configure login security (blocking)
- Set system clock

---

## 🔐 Security Features Implemented
- Console and VTY password protection
- Encrypted passwords
- Login blocking after failed attempts
- Banner message for unauthorized access warning
